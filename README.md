# QA Automation Framework - Cypress

End-to-end test automation project built with Cypress using the Page Object Model (POM) pattern.

This project demonstrates automated testing for a sample e-commerce application (SauceDemo), covering core user flows such as authentication, cart management, and checkout.

## Tech Stack

- Cypress
- JavaScript
- Node.js
- Page Object Model (POM)

## Test Scenarios

The following scenarios are automated:

- Successful login
- Invalid login validation
- Add product to cart
- Complete checkout flow
